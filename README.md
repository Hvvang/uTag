## Required software:
  - `git`;
  - any `C++ compiler` supported `17 standard` and higher;
  - `make`;
  - `cmake` `v3.17` and higher.

## Installation
>**1) git clone --recursive [`https://github.com/Hvvang/uTag.git`](https://github.com/Hvvang/uTag.git)**  
>**2) `cd uTag && cmake . -Bbuild && cmake --build ./build`**  
>**3) `./utag [dir]`**

## Tested Systems:
